== Description ==
Simple tool to show social networks statistic by url. Support this site: Tweeter, Linkedin, Facebook, Reddit, Stumbleupon, Delicious, Pinterest, Google Plus. 
- 

== Installation ==

1.  Upload Social_counter folder to plugins folder.

2.  Go to Plugins menu and active.

3.  If you want to show only some page social networks statistic add this shorcode to post/page end: [social_counter][/social_counter].

4.  Also is widget, which show page statistic in sidebar.
